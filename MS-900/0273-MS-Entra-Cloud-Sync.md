# Microsoft Entra Cloud Sync

Cloud Sync is a lightweight identity synchronization tool that replaces traditional Azure AD Connect in certain scenarios.

## Advantages

- Simplified deployment
- Agent-based synchronization
- High availability
- Cloud-managed configuration

## Use Case

A mid-sized company migrates to Cloud Sync for simplified management across multiple branch offices.

## Example

IT installs Cloud Sync agent in each region for resilient identity replication.
