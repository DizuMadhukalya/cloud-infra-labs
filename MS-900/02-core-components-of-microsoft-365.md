# Core Components of Microsoft 365

Microsoft 365 consists of multiple integrated components:

## Productivity
- Word, Excel, PowerPoint
- Outlook and OneNote

## Collaboration
- Microsoft Teams
- SharePoint Online
- OneDrive for Business

## Security
- Microsoft Defender
- Identity protection via Azure AD

## Device Management
- Microsoft Intune
- Endpoint security and compliance

These components work together to provide a unified digital workplace.
