# On-Premises Synchronization

On-premises synchronization ensures identity data flows from local AD to Microsoft Entra ID.

## Synchronization Methods

- Password Hash Sync
- Pass-through Authentication
- Federation

## Benefits

- Single Identity
- Reduced password fatigue
- Central identity governance

## Real-Life Example

A hospital uses local AD for workstation logins and syncs identities to Microsoft 365 for secure email and Teams communication.
