# Role of Microsoft Graph in Microsoft 365

Microsoft Graph is the unified API that connects data across Microsoft 365 services.

## What is Microsoft Graph?
- A gateway to data from Microsoft 365 services
- Provides access to users, emails, calendars, files, Teams, and more
- Enables developers and services to interact with Microsoft 365 data securely

## Role of Microsoft Graph
- Connects Microsoft 365 apps like Outlook, Teams, OneDrive, and SharePoint
- Enables intelligent services by providing contextual data
- Supports automation, analytics, and AI-powered experiences

## Examples of Data Accessed
- User profiles and identities
- Emails and calendars
- Files and documents
- Teams chats and meetings

Microsoft Graph is the foundation for intelligent and connected experiences across Microsoft 365.
