# CASB in Microsoft Defender for Cloud Apps

CASB (Cloud Access Security Broker) acts as a security control point between users and cloud applications.

It monitors and controls access to cloud apps.

---

## CASB Core Functions

### Visibility
Shows cloud app usage.

Example:
Employee using unauthorized file-sharing apps.

---

### Threat Protection

Detects:

- Suspicious login locations
- Malware uploads
- Unauthorized access

---

### Data Protection

Prevents sensitive data leaks.

Example:
Block upload of confidential files.

---

## Real-Life Example

Employee uploads company financial data to personal Dropbox.

CASB detects upload and blocks it.

Result:
Data leak prevented.

---

## Benefits

- Control Shadow IT
- Protect sensitive data
- Monitor cloud usage
- Detect threats early
