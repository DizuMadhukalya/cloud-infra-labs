# Introduction to Microsoft Defender XDR

Microsoft Defender XDR (Extended Detection and Response) is a unified security platform that integrates multiple Microsoft security solutions into a single coordinated defense system.

It provides:

- Centralized incident management
- Cross-domain threat correlation
- Automated investigation and response
- Unified security portal

---

## What XDR Means

Instead of security tools working in isolation (email, endpoint, identity), Defender XDR:

Correlates signals across:
- Endpoints
- Identities
- Email
- Cloud apps
- SaaS platforms

---

## Real-Life Use Case

A phishing email infects a user device:
- Defender for Office detects email
- Defender for Endpoint detects suspicious process
- Defender for Identity flags risky login

XDR correlates all alerts into a single incident.

---

## Example

Security team sees one unified incident instead of three separate alerts.
