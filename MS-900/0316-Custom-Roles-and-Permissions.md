# Custom Roles in Microsoft Entra

Custom roles allow fine-grained permission control.

## Process

1. Define role name and description.
2. Select permissions (e.g., manage groups only).
3. Define assignable scope.
4. Assign to user or group.

---

## Real-Life Example

Organization creates:
"Group Manager – Marketing"

Allows:
- Manage only marketing groups
Not allowed:
- Modify tenant-wide settings
