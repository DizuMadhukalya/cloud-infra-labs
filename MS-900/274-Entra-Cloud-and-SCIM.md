# Microsoft Entra Cloud and SCIM

SCIM (System for Cross-domain Identity Management) automates user provisioning to SaaS apps.

## Benefits

- Automatic user creation/deletion
- Reduced manual work
- Consistent access management

## Real-Life Use Case

When employee joins company, Entra automatically provisions account in Salesforce and ServiceNow using SCIM.

## Example

When employee leaves, SCIM deprovisions access instantly.
