# Global Secure Access Apps

Global Secure Access protects both:

- SaaS applications
- Microsoft 365 services
- Private internal applications
- Third-party cloud apps

---

## Application Types

1. Public SaaS Apps (Salesforce, ServiceNow)
2. Microsoft 365 Apps
3. Internal Web Applications
4. Hybrid Cloud Workloads

---

## Real-Life Use Case

A company uses:
- Microsoft 365
- SAP hosted internally
- Salesforce cloud CRM

Global Secure Access ensures:
Identity-based access across all three.
