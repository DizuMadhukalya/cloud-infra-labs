# Microsoft Entra Internet Access Deployment

Entra Internet Access secures internet-bound traffic using identity-driven policies.

## Deployment Steps

1. Enable Global Secure Access.
2. Configure traffic forwarding method.
3. Define web filtering policies.
4. Integrate Conditional Access.
5. Assign users or groups.

---

## Traffic Forwarding Options

- Endpoint client
- Network-based forwarding
- Proxy integration

---

## Real-Life Scenario

Remote employee browsing:
- SaaS tools
- Web portals
- Cloud applications

Traffic flows securely through Entra Internet Access.

---

## Example

User tries to access malicious website.
