Microsoft Planner is a task management tool integrated with Microsoft Teams and Microsoft 365 Groups for collaborative task tracking.

Features:

- Kanban-style board

- Buckets for categorization

- File attachments via SharePoint

- Outlook notifications

- Progress charts

Use cases:

- Team task tracking

- Agile work management

- Project collaboration

MS-900 relevance:
Highlights how Microsoft 365 Groups enable shared resources like Planner, mailbox, and calendar.
