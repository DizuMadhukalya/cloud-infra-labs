# Inter-Directory Provisioning in Hybrid Identities

Inter-directory provisioning ensures users and groups created in on-prem Active Directory are synchronized to Microsoft Entra ID.

## How It Works

- Users created in on-prem AD
- Azure AD Connect syncs objects
- Changes reflect automatically in cloud
