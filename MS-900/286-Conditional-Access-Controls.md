# Access Controls in Conditional Access

Access Controls determine what happens after evaluation.

## Block Access
Complete denial.

## Grant Access
Requires:
- MFA
- Compliant device
- Hybrid joined device
- Passwordless authentication

## Session Controls
- Sign-in frequency
- Continuous Access Evaluation
- App-enforced restrictions

---

## Real-Life Scenario

Guest user accessing SharePoint:
Grant only if:
- MFA completed
- Device compliant
