# Microsoft 365 Copilot Subscription

Microsoft 365 Copilot is offered as an add-on subscription for eligible Microsoft 365 plans.

## Subscription Overview
- Copilot is not included by default in Microsoft 365
- Requires an existing Microsoft 365 license
- Purchased as an additional paid add-on

## Eligibility
- Available for select enterprise and business plans
- Requires Microsoft 365 Apps and cloud services
- Organizations must meet security and compliance requirements

## Licensing Model
- Per-user subscription
- Centralized management through Microsoft 365 admin center
- Designed for enterprise-scale deployment

Microsoft 365 Copilot subscription enables organizations to unlock AI-powered productivity while maintaining security and compliance.
