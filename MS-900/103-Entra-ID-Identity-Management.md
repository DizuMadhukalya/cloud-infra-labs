# Microsoft Entra ID (Azure AD) Identity Management

Microsoft Entra ID provides identity and access management for Microsoft 365.

## Features
- User authentication
- Single Sign-On (SSO)
- Multi-Factor Authentication (MFA)
- Conditional Access
- Role-Based Access Control (RBAC)

## Use Case
Employees sign in once and access Outlook, Teams, and SharePoint without multiple passwords.

## Example
Admin enforces MFA for finance department using Conditional Access policies.
