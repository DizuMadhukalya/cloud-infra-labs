# Microsoft Copilot Chat

Microsoft Copilot Chat is a conversational AI experience integrated across Microsoft 365 apps.

## What is Microsoft Copilot Chat?
- An AI-powered chat interface for interacting with Microsoft 365 data
- Available in apps like Teams, Outlook, Word, Excel, and the Microsoft 365 portal
- Allows users to ask questions in natural language

## Key Capabilities
- Ask questions about emails, meetings, and documents
- Generate summaries and insights from conversations and files
- Draft content such as emails, reports, and presentations
- Provide contextual answers based on organizational data

## How Copilot Chat Works
- Uses Large Language Models (LLMs) for language understanding
- Pulls relevant data securely via Microsoft Graph
- Respects user permissions, security, and compliance policies

## Benefits of Copilot Chat
- Improves productivity and reduces manual effort
- Helps users find information faster
- Enhances collaboration and decision-making

Microsoft Copilot Chat acts as a centralized AI assistant that supports users throughout their daily Microsoft 365 workflows.
