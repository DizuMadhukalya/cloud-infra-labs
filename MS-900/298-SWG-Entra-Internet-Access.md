# Secure Web Gateway (SWG)

Secure Web Gateway protects users from web-based threats.

It provides:

- URL filtering
- Malware inspection
- TLS inspection
- Data loss protection enforcement

---

## Why SWG is Important

Traditional firewall cannot protect remote users outside corporate network.

SWG:
- Protects traffic anywhere
- Applies identity-aware filtering

---

## Real-Life Example

Employee downloads infected file from suspicious website.
SWG blocks download before infection.
