# How Microsoft Teams Promotes Collaboration and Enhances Teamwork

Microsoft Teams is the central hub for teamwork in Microsoft 365.

## Core Components
- Teams and Channels
- Chat and Instant Messaging
- File sharing via SharePoint
- Meetings and calls
- Copilot assistance

## Features
- Persistent chat history
- Threaded conversations
- Real-time file collaboration
- Mentions and notifications
- Copilot summaries

## Use Case
HR team uses a Recruitment channel, chat for coordination, and Copilot for summaries.

## Example
Developers, designers, and managers collaborate transparently in channels.
