# Continuous Access Evaluation (CAE)

CAE ensures that access decisions are not static.

Access is re-evaluated when:

- User risk changes
- Device becomes non-compliant
- Password reset occurs
- Account disabled

---

## Why CAE Matters

Traditional tokens valid for hours.
CAE can revoke access instantly.

---

## Real-Life Example

User laptop flagged as compromised.
Access to SharePoint revoked immediately.
