# SaaS Security Posture Management (SSPM)

SSPM evaluates and improves security configuration of SaaS applications.

Examples:
- Microsoft 365
- Salesforce
- Google Workspace

---

## What SSPM Detects

- Weak configurations
- Excessive permissions
- Disabled security controls

Example:
Multi-factor authentication disabled.

---

## Real-Life Example

Company admin disables audit logging in Microsoft 365.

SSPM detects configuration risk.

Admin re-enables logging.

Result:
Improved security visibility.

---

## Benefits

- Prevent misconfiguration risks
- Improve cloud security posture
- Reduce attack surface
