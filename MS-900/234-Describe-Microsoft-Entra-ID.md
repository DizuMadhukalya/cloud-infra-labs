# Describe Microsoft Entra ID

Microsoft Entra ID is the identity and access management solution for Microsoft 365 and Azure resources.

## Resources Protected
- Internal resources: SharePoint sites, Teams, Exchange, OneDrive
- External resources: SaaS apps, partner portals, third-party apps

## How It Simplifies Access
- Single Sign-On (SSO)
- Centralized authentication
- Conditional Access policies
- Role-based access control

## Example
User logs in once and accesses Teams, Outlook, and SharePoint without re-authentication.
