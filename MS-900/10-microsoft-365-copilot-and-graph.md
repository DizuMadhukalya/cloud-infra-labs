# Large Language Models (LLMs) in Microsoft 365 Copilot

Microsoft 365 Copilot uses Large Language Models (LLMs) to understand and generate human-like responses.

## What are LLMs?
- Advanced AI models trained on large datasets
- Capable of understanding natural language
- Generate text, summaries, and insights

## Role of LLMs in Copilot
- Interpret user prompts written in natural language
- Generate emails, documents, summaries, and insights
- Work together with Microsoft Graph data

## How LLMs and Graph Work Together
- LLMs handle language understanding and generation
- Microsoft Graph provides real-time organizational context
- Combined approach ensures accuracy and relevance

LLMs enable Copilot to act as an intelligent assistant rather than a simple automation tool.
