# Microsoft 365 Copilot and Microsoft Graph

Microsoft 365 Copilot uses Microsoft Graph to deliver personalized and context-aware assistance.

## How Copilot Uses Microsoft Graph
- Retrieves relevant organizational data securely
- Understands user context such as emails, meetings, and documents
- Provides intelligent suggestions based on real work data

## Examples of Copilot Usage
- Summarizing emails and meetings
- Drafting documents using existing files
- Answering questions using organizational knowledge

## Data Security
- Copilot respects Microsoft 365 security, permissions, and compliance
- Only data the user is authorized to access is used
- No data is used without proper access controls

Microsoft Graph acts as the data backbone that enables Copilot to be relevant, secure, and productive.
