# Threat Detection Using Real-Time Analysis and Data Intelligence

Defender for Identity detects attacks in real time.

---

Reconnaissance detection

Detects attackers scanning network.

Example:

Attacker tries to identify domain admins.

---

Compromised credentials detection

Detects stolen credential usage.

Example:

Login from unusual location.

---

Lateral movement detection

Detects attacker moving across systems.

Example:

Using compromised account to access multiple servers.

---

Domain dominance detection

Detects full domain takeover attempts.

Example:

Attacker attempts to control Active Directory.

---

Conclusion:

Provides early detection of identity attacks.
