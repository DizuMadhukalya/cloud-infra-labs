# Azure Virtual Desktop (AVD)

Azure Virtual Desktop is a desktop and app virtualization service running on Azure.

## Key Features
- Multi-session Windows 10/11
- Remote app streaming
- Flexible scaling
- Pay-as-you-use pricing
- Advanced networking control
- FSLogix profile containers

## Use Case
A university provides virtual lab environments to students through AVD during practical sessions.

## Example
Developers access GPU-enabled virtual machines for testing without local hardware requirements.
