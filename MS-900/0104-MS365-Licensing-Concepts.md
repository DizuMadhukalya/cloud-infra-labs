# Licensing in Microsoft 365

Licenses determine which services and features users can access.

## License Types
- Business (Basic, Standard, Premium)
- Enterprise (E3, E5)
- Add-ons (Copilot, Audio Conferencing)

## Assignment Methods
- Direct assignment
- Group-based licensing

## Use Case
HR group automatically receives Teams and Exchange licenses via group membership.

## Example
New employee added to “Sales” group automatically gets required apps.
