# Deploy from Local Source with Configuration Manager

Configuration Manager allows IT admins to download Microsoft 365 Apps installation files once and distribute them across the local network.

## Benefits
- Saves internet bandwidth
- Centralized control
- Scheduled deployment
- Integration with existing infrastructure

## Use Case
Head office downloads Office binaries and branch devices install from local DP without internet usage.

## Example
IT schedules Office deployment overnight to avoid work-hour disruptions.
