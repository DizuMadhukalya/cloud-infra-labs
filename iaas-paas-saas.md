# IaaS, PaaS and SaaS

Cloud services are divided into three main categories.

---

## 1. Infrastructure as a Service (IaaS)

Provides virtualized hardware over the internet.

Examples:
- Azure Virtual Machines  
- Virtual Networks  
- Storage accounts  

User controls:
- Operating system  
- Applications  
- Data  

---

## 2. Platform as a Service (PaaS)

Provides a platform for developing and hosting applications.

Examples:
- Azure App Service  
- Azure SQL Database  

User controls only:
- Application  
- Data  

---

## 3. Software as a Service (SaaS)

Ready-to-use software hosted in the cloud.

Examples:
- Microsoft 365  
- Outlook  
- Teams  

User only uses the application.

---

## Summary

| Model | Control Level |
|-----|---------------|
| IaaS | Most control |
| PaaS | Moderate control |
| SaaS | Least control |
