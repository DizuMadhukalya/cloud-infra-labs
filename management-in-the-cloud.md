# Management in the Cloud

Management in the cloud refers to how customers manage their own resources hosted in Azure.

---

## What is Management in the Cloud?

It includes tasks performed by users such as:

- Creating virtual machines  
- Configuring networks  
- Monitoring applications  
- Managing identities  

---

## Tools Used

Azure provides multiple management options:

- Azure Portal  
- Azure CLI  
- PowerShell  
- REST APIs  
- ARM Templates  

---

## Benefits

- Full control over resources  
- Automation capabilities  
- Real-time monitoring  
- Flexible configuration  

---

## Conclusion

While Azure manages the infrastructure, customers manage their own cloud resources using powerful tools provided by the platform.
