# Shared Responsibility Model

In cloud computing, security and management responsibilities are shared between the cloud provider and the customer.

---

## What is Shared Responsibility?

Microsoft Azure is responsible for:

- Physical data centers  
- Network infrastructure  
- Hardware  
- Core platform security  

Customers are responsible for:

- Their data  
- User access  
- Configurations  
- Applications  

---

## Why it Matters

This model clarifies:

- Who manages what  
- What customers must secure  
- What Azure automatically handles  

Understanding this is critical for cloud security.
